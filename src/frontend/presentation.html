Câu Hỏi Thường Gặp

Chuẩn bị cho phần Q&A

Q1: Tại sao chọn Vanilla JavaScript thay vì React/Vue?

A: Hiệu năng tốt hơn (không overhead), bundle size nhỏ hơn (loading nhanh), học được JavaScript thuần (hiểu sâu hơn), dự án không quá phức tạp.

Q2: Làm thế nào để tích hợp VietQR?

A: Sử dụng VietQR API (img.vietqr.io), tự động điền amount + transaction code + account name. Bank info từ .env (bảo mật).

Q3: Làm thế nào để scale khi có nhiều user?

A: Ngắn hạn: Redis caching, CDN cho static files, connection pool. Dài hạn: Load balancer, database replication, microservices, message queue.
